# ☕📚 La Libreria Cafe

A responsive café and bookstore themed website developed using HTML, CSS, and Bootstrap 5.

## Features

* Responsive Bootstrap Layout
* Interactive Carousel
* Modern Navigation Bar
* Book Collection Cards
* Café Menu Section
* Reservation Form
* Testimonials Section
* Hover Animations

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* GitHub Pages
